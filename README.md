# Async User Manager System 🚀

Проект, созданный в рамках обучения на курсе **OpenClassrooms: Learn Programming With JavaScript**. Это полноценное мини-приложение для управления данными пользователей с имитацией серверной задержки.

### 🛠 Использованные технологии:
* **JavaScript (ES6+)**: Классы, стрелочные функции, деструктуризация.
* **Asynchronous JS**: Использование `async/await` и `Promises` для работы с задержками.
* **DOM Manipulation**: Связь бизнес-логики с HTML-интерфейсом.
* **Web Storage API**: Сохранение данных в `localStorage`.

### ⚡️ Ключевые особенности:
* **Guard Clauses**: Валидация данных (возраст 18+, длина имени) до начала выполнения операций.
* **Error Handling**: Обработка исключений через блоки `try...catch...finally`.
* **Persistence**: Данные не пропадают после перезагрузки страницы.

### 📖 Чему я научился:
В ходе разработки этого проекта я закрепил навыки написания чистого кода, работы с контекстом `this` в классах и управления состоянием интерфейса во время асинхронных запросов.
